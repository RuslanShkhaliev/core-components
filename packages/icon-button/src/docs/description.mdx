Компонент может использоваться самостоятельно или в составе более сложных компонентов: Calendar, FilterTag.

## Примеры

### Размеры
Рекомендуется использовать S, XS и XXS размеры, в зависимости от контекста.

```jsx live
<Container>
    <Row>
        <Col>
            <IconButton
                view='primary'
                size='s'
                icon={StarMIcon}
                style={{ border: '1px solid rgba(0,0,0,0.1) ' }}
            />
        </Col>
        <Col>
            <IconButton
                view='primary'
                size='xs'
                icon={StarMIcon}
                style={{ border: '1px solid rgba(0,0,0,0.1) ' }}
            />
        </Col>
        <Col>
            <IconButton
                view='primary'
                size='xxs'
                icon={StarMIcon}
                style={{ border: '1px solid rgba(0,0,0,0.1) ' }}
            />
        </Col>
    </Row>
</Container>
```

### Состояние загрузки
С помощью свойства loading можно отобразить состояние загрузки. Минимальное время отображения лоадера - 500мс, чтобы при быстрых ответах от сервера кнопка не «моргала».

```jsx live
<Container>
    <Row>
        <Col>
            <IconButton
                view='primary'
                icon={StarMIcon}
                loading={true}
            />
        </Col>
        <Col>
            <IconButton
                view='secondary'
                icon={StarMIcon}
                loading={true}
            />
        </Col>
        <Col>
             <IconButton
                view='tertiary'
                icon={StarMIcon}
                loading={true}
            />
        </Col>
        <Col>
            <IconButton
                view='negative'
                icon={StarMIcon}
                loading={true}
            />
        </Col>  
    </Row>
</Container>
```
